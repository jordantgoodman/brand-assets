POPS — Font Comparison

Source Sans 3 vs Public Sans vs Inter vs Avenir Next · Side-by-side in brand colors and hierarchy · March 2026

Recommendation

#1 Recommended
Source Sans 3
Best Windows ClearType hinting of any free sans-serif. Superior rendering at 11–13px in dense compliance tables — exactly where POPS materials live. Straight geometric "l" with no softness. Source Code Pro companion for part numbers. Built by Paul Hunt at Adobe with explicit focus on screen readability across platforms. The technical choice.
Best Technical Fit
#2 Strong Alternative
Public Sans
The U.S. government's own typeface — built by USWDS for federal digital services. When a CDCR procurement officer reads a POPS bid package set in Public Sans, it subconsciously reads as "official." Maximum institutional authority. Slightly less optimized Windows hinting than Source Sans 3, and no monospace companion. The brand-alignment choice.
Strongest Brand Signal
Reference — Current Standard
Inter
Clean and well-built, but documented rendering issues below 12px on Windows without specific ClearType settings. Personality reads as "tech/UI" rather than "institutional authority." Used by Figma, Notion, GitLab — signals modern software, not government infrastructure. Solid fallback but not the best primary for this audience.
Reference Only
Reference — Compatibility Baseline
Avenir Next
Beautiful typeface with institutional pedigree (Frutiger), but proprietary licensing, Mac-only system availability, no Google Fonts, and Windows fallback issues make it impractical for a brand that delivers bid packages to government procurement officers on Windows. Included for visual comparison only.
Reference Only

Critical Glyph Comparison — l / I / 1

The lowercase "l" is a litmus test for institutional authority. Curved tails read as soft. Corrections materials need straight, no-nonsense letterforms.

Source Sans 3 Recommended

Straight geometric "l" — clean vertical stroke. No decorative features. Strong disambiguation between all three characters. Reads as authoritative at every size.

Construction
The POPS One communication booth is engineered for correctional facility environments requiring secure, private video teleconference capabilities. All-steel welded construction with 16-gauge panels, ligature-resistant hardware, and tamper-proof fasteners. Integrated HVAC system maintains comfortable operating temperatures in all facility zones.
Compliance
Meets ASTM E84 Class A fire rating, ASTM E90 sound transmission, and California Building Code Chapter 16 requirements. All hardware is ligature-resistant per facility security standards.

Bid Document Simulation

The real test — how each font looks in a compliance-heavy bid package section at 10–11px body text

Source Sans 3 Recommended
Section 12 64 13 — Prefabricated VEST Booths
2.12 Hardware
A. Door Hardware: Provide commercial-grade, ligature-resistant door hardware meeting the following requirements:
1. Hinges: Stanley CECB179 or approved equal. Heavy-duty, full mortise, ball bearing, ligature-resistant. 4-1/2" × 4-1/2", US26D finish.
2. Pull Plate: Ives 8302 push/pull plate. 16" × 4", 0.050" thick, US32D finish.
3. Closer: LCN 4040XP or approved equal. Cast iron body, full cover, delayed action.
B. All fasteners shall be tamper-proof security type, minimum 12-24 thread, stainless steel. Provide security bit with each booth delivery.
Reference: CDCR VEST Booths RFP, Section 12 64 13, Part 2 — Products
